This folder is used to store the three candidate YOLO models for tracking experiments.

The model weights are not included in the GitHub repository due to file size.

Required files:
- yolo26n_RC1C2_best.pt
- yolo26s_RC1C2_best.pt
- yolo26m_RC1C2_best.pt

They can be copied from:

```
Tennis_dataset/experiment/runs/yolo26n_RC1C2/weights/best.pt
Tennis_dataset/experiment/runs/yolo26s_RC1C2/weights/best.pt
Tennis_dataset/experiment/runs/yolo26m_RC1C2/weights/best.pt
```
