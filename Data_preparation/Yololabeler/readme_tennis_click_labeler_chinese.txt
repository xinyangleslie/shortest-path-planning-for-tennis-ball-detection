网球点击标注工具说明

==================================================
1. 工具用途
==================================================
这个工具用于通过鼠标点击的方式标注网球。
程序会在每个点击点周围生成固定大小的小框，
并将标注结果导出为 YOLO 格式。

核心思路：
- 一次打开一张图片
- 用鼠标左键点击每个网球的大致中心
- 按下 'd' 生成固定大小的标注框
- 按下 's' 保存标签并进入下一张图片
- 程序会自动导出对应的 YOLO 格式 .txt 文件

==================================================
2. 运行环境
==================================================
需要 Python 3.x 环境

所需 Python 库：
- opencv-python

如果还没有安装 OpenCV，可以执行：
pip install opencv-python

==================================================
3. 文件准备
==================================================
将所有需要标注的图片放入同一个文件夹中。

支持的图片格式：
- .jpg
- .jpeg
- .png
- .bmp
- .webp

示例：
images/
    img001.jpg
    img002.jpg
    img003.png

==================================================
4. 如何运行
==================================================
在包含以下文件的文件夹中打开终端或命令提示符：
- tennis_click_labeler.py

运行命令：

python tennis_click_labeler.py --image_dir ./images --output_dir ./fixed_box_output --class_id 0

参数说明：
--image_dir
    输入图片所在文件夹

--output_dir
    保存标签和可视化结果的输出文件夹

--class_id
    YOLO 的类别编号
    如果标注的是 tennis ball，一般设置为 0

Windows 示例：
python tennis_click_labeler.py --image_dir "C:\Users\YourName\Desktop\images" --output_dir "C:\Users\YourName\Desktop\fixed_box_output" --class_id 0

==================================================
5. 鼠标与键盘操作
==================================================
鼠标操作：
- 左键：
    在网球位置添加一个红点
- 右键：
    删除最近一次添加的点

键盘操作：
- d
    根据所有点击点生成标注框
- s
    保存当前标签并进入下一张图片
    注意：保存前需要先按 'd'
- u
    撤销最后一个点
    如果当前已经在 review 模式，会先回到点击模式
- r
    重置当前图片的所有标注
- n
    跳过当前图片且不保存
- q
    退出程序

==================================================
6. 推荐标注流程
==================================================
每张图片建议按照以下步骤进行：

步骤 1：
在程序中打开当前图片

步骤 2：
对每一个网球点击一次左键
尽量点击在球的中心附近

步骤 3：
如果某个点点错了，可以使用：
- 鼠标右键，或者
- 按键 'u'

步骤 4：
当所有点都添加完成后，按下：
d

程序会根据点击点生成固定大小的标注框

步骤 5：
检查生成的框是否大致合理

步骤 6：
如果确认没有问题，按下：
s

程序会保存 YOLO 标签文件，并自动进入下一张图片

==================================================
7. 输出文件
==================================================
程序会在 output_dir 中自动创建两个文件夹：

1) labels/
    存放 YOLO 标注文本文件
    示例：
    img001.txt

2) vis/
    存放可视化结果图片
    这些图片会显示点击点和生成后的框
    示例：
    img001.jpg

==================================================
8. YOLO 标签格式
==================================================
保存的 .txt 文件中，每一行格式如下：

class_id x_center y_center width height

所有数值都已经归一化到 [0, 1] 区间。

示例：
0 0.512500 0.635417 0.010938 0.014583

==================================================
9. 标注框大小逻辑
==================================================
这个工具并不会自动检测目标，
而是根据鼠标点击的位置生成固定大小的框。

框大小规则：
- 图像中大多数区域：
    使用较小的框
- 图像最下方 1/4 区域的点：
    使用稍大的框

原因：
因为图像下方的物体在透视下通常会显得稍微大一些。

==================================================
10. 注意事项
==================================================
1. 每个网球只点击一次
2. 尽量点在球的中心附近
3. 按 's' 保存前必须先按 'd'
4. 如果按 'n' 跳过当前图片，则不会生成标签文件
5. 如果按 'q' 退出，程序会立即停止
6. 这个工具是为了加快人工标注，不是自动检测程序

==================================================
11. 代码思路简要解释
==================================================
这段代码的工作流程如下：

- 先读取输入文件夹中的所有图片
- 一次显示一张图片
- 每次鼠标左键点击会记录一个点
- 程序再把这个点转换成一个固定大小的标注框
- 然后把这些框转换成 YOLO 格式
- 最后将标签文件和预览图保存到输出文件夹中

所以整体逻辑可以总结为：
鼠标点击 -> 点坐标 -> 固定框 -> YOLO 标签文件

==================================================
12. 常见问题
==================================================
问题：
No images found

可能原因：
图片文件夹路径错误，或者文件夹中没有图片

问题：
程序提示 "Please press 'd' first."

原因：
你在生成框之前就尝试保存了

问题：
没有弹出 OpenCV 窗口

可能原因：
OpenCV 没有正确安装，或者当前使用的 Python 环境不对

13. 建议给使用者的提示
==================================================
标注时建议：
- 每个网球中心点击一次
- 按 'd' 生成框
- 检查框是否合理
- 按 's' 保存
- 如果某个点错了，可以用右键或按 'u' 撤销
